# PAV.GROUP Landing Page

Landing page premium desenvolvida para apresentar a proposta da PAV.GROUP com foco em clareza, sofisticação visual, experiência fluida e alta conversão.

O projeto combina uma narrativa visual baseada em cards animados por scroll, transições suaves e um footer final minimalista, criando uma experiência imersiva sem perder performance, usabilidade e objetividade.

## Visão geral

A página foi criada para comunicar valor de forma rápida e elegante.  
A experiência começa com uma sequência de cards que apresentam os pilares da oferta e evolui para uma transição visual onde os cards se unem, formam um retângulo branco e revelam o footer final.

O foco principal do projeto é equilibrar estética premium, performance e conversão.

## Principais recursos

- Layout premium com fundo escuro e cards em destaque
- Animações controladas pelo scroll
- Transição dos cards para um retângulo branco expansivo
- Footer final com identidade visual, chamada e CTAs

- Estrutura pensada para alta conversão
- Interface limpa, minimalista e objetiva
- Animações suaves com foco em performance

## Seções do projeto

### Cards animados

A primeira parte da página apresenta quatro cards principais:

1. Diagnóstico da oferta  
2. Arquitetura da página  
3. Direção visual premium  
4. Design responsivo  

Cada card comunica uma etapa importante da proposta, com títulos fortes, textos curtos e numeração visual em destaque.

### Transição visual

Após a sequência dos cards, os elementos se juntam no centro da tela e formam um retângulo branco.  
Esse retângulo se expande até ocupar toda a viewport, funcionando como ponte visual para o footer.

### Footer

O footer final apresenta:

- Logo PAV.GROUP
- Texto legal
- Link de política de privacidade
- Chamada principal
- Botão de aplicação para Sprint
- Botão de contato

A entrada dos elementos do footer acontece antes da animação do retângulo terminar, deixando a transição mais fluida e natural.

## Performance e UX

As animações foram planejadas para enriquecer a jornada do usuário sem prejudicar a navegação.  
O objetivo é manter a página leve, rápida e agradável, evitando efeitos desnecessários ou interações que confundam o visitante.

Também foram consideradas boas práticas como:

- Scripts otimizados
- Código limpo e reutilizável
- Animações suaves
- Responsividade em diferentes tamanhos de tela
- Compatibilidade com usuários que preferem reduzir movimento
